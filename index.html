04 — Who Built This

From hardware counter to software shop.

Ten years ago I opened The Computer Services in Vijaypur with a screwdriver set and a soldering iron. The work was simple: fix the PC, recover the photos, set up the printer. It still is, most days.

What changed is what happens after the hardware is fixed. A customer needs a document scanned, a PF number calculated, a photo resized to a government portal's exact pixel count — and the nearest service that does it properly is an hour away, on a road with patchy signal. So I started building the tool myself, in plain HTML and JavaScript, designed to run with or without the internet.

Eighteen tools later, that habit has become the actual shop: a counter that fixes hardware and ships its own software, built one customer problem at a time.
